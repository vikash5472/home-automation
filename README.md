# home-automation Nodejs API

## Description

Based on [ExpressJs](https://expressjs.com/) framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# run all services for development
# watch mode
$ npm start:dev

# production mode
$ npm start:prod
```

## technology used

### nodejs
### expressjs framework
### mongodb
